=================================
USER CONFIGURATION REQUIREMENTS
=================================

The platform must allow users to select:

1. Project Name

2. Technology Stack
   - Java
   - Python

3. Automation Framework
   - Rest Assured
   - Selenium
   - Playwright

4. Output Directory

Example:

D:\AI-SDLC-Projects

The application shall create the folder automatically if it does not exist.

The generated project shall be saved under:

<Output Directory>\<Project Name>_<Timestamp>

Example:

D:\AI-SDLC-Projects\HotelBooking_20260618_153000

All generated artifacts and source code shall be written to this directory.

Generated artifacts include:

- Requirement Specification
- User Stories
- Acceptance Criteria
- BDD Scenarios
- Test Cases
- Automation Framework
- Generated Source Code
- Final Report

The Streamlit UI shall display the generated project location after execution.

The Streamlit UI shall provide:
- Open Folder button
- Download ZIP button
